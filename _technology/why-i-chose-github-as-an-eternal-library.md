For the first time in years, I decided not to use WordPress. My main motivation was to find a platform that will outlive my time and preserve my thoughts.

## The problem with traditional hosting

Blogs hosted on WordPress, Squarespace, or custom servers share a common vulnerability: they depend on recurring payments. When a credit card expires, when a hosting company pivots, when a domain registrar changes policies—content disappears. The internet is not as permanent as it feels.

## GitHub as an institution

I see GitHub playing the role of that eternal library—not just for code, but also for progressive ideas. A repository does not expire when a hosting plan lapses. It persists as long as the institution behind it does. And GitHub, backed by Microsoft and serving as the world's de facto code archive, has a durability that few platforms can match.

## The format is the commitment

Static HTML, Markdown files, and a Git history. No database to corrupt. No plugin to break. No theme marketplace to navigate. The simplicity is the point. A site that can be rebuilt from a single `git clone` command is a site that can survive anything short of civilizational collapse.

## What this enables

By choosing this platform, I am free to think in decades. Each essay joins a growing archive that requires no maintenance, no updates, no vigilance. It simply persists.
